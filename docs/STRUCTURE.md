"Для реалізації обрано архітектурний паттерн Clean Architecture з використанням фреймворків Angular (Client) та NestJS (Server). На початковому етапі розробки застосовано підхід модульного моноліту для забезпечення швидкої ітерації MVP, з можливістю подальшої декомпозиції на окремі мікросервіси через чітку ізоляцію бізнес-модулів (Auth, Volunteer, Users)."

[x] Налаштував середовище розробки на macOS Monterey без Docker.

[x] Реалізував схему даних для волонтерських запитів з підтримкою геолокації.

[x] Спроектував API за принципом Code-First, що гарантує 100% відповідність типів TypeScript та GraphQL-схеми.

[x] Налагодив автоматичне створення структури БД через сутності (Entities).

[x] Оптимізував Git-репозиторій, вичистивши історію від залежностей node_modules.
